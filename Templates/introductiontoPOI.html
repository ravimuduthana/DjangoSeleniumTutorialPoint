Interfaces & Classes

# Interface Description Class
1 Workbook A workbook represents a Microsoft Excel file. It can be used for creating and maintaining the spreadsheet. A workbook may contain many sheets HSSFWorkbook- Is a class representation of XLS file
XSSFWorkbook- Is a class representation of XLSX file
2 Sheet A sheet refers to a page in a Microsoft Excel file that contains the number of rows and columns HSSFSheet-Is a class representation of XLS file
XSSFSheet-Is a class representation of XLSX file
3 Row A row represents a collection of cells, which is used to represent a row in the spreadsheet. HSSFRow-Is a class representation of XLS file
XSSFRow-Is a class representation of XLSX file
4 Cell A cell is indicated by a row and column combination. Data entered by a user is stored in a cell. Data can be of the type such as string, numeric value, or formula. HSSFCell-Is a class representation of XLS file
XSSFCell-Is a class representation of XLSX file
