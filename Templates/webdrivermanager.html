WebDriverManager

WebDriverManager is a library which allows us to automate the management of the drivers (e.g. iedriver, chromedriver, geckodriver, etc.) required by selenium web driver.
Go to maven repository ,find stable version and get the dependency from https://mvnrepository.com/artifact/io.github.bonigarcia/webdrivermanager or add dependency from below into your pom.xml file
Use these code in the script to initiate the driver using webdriver manager 
               
                //set webdrivermanager for chrome
                WebDriverManager.chromedriver().setup();
                ChromeDriver driver=new ChromeDriver();
            
        
               
        
        package Scripts;

        import org.openqa.selenium.WebDriver;
        import org.openqa.selenium.chrome.ChromeDriver;

        import io.github.bonigarcia.wdm.WebDriverManager;

        public class Webdrivermanager {
	    public static void main(String[] args) throws InterruptedException {
		
		
		//With webdriver manager,initiating the driver and opening the website
		
		WebDriverManager.chromedriver().setup();
		ChromeDriver driver1=new ChromeDriver();
		driver1.get("http://www.testautomationpoint.com/");
		
	}

}
